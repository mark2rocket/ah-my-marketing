## 브랜드 코드

**코드:** `{BrandCode}` (예: BrandA)

---

## 캠페인 네이밍 규칙

```
{BrandCode}_{목표코드}_{YYMM}_{메모(선택)}
```

**예시:** `BrandA_CPI_2604`

### 사용 목표 코드

| 코드 | 설명 | 사용 여부 |
|------|------|----------|
| CPI | 앱 설치 | |
| CON | 전환 | |
| TR | 트래픽 | |
| LD | 리드 | |
| AW | 인지 | |

---

## 광고세트 네이밍 규칙

```
{타겟타입}_{성별}{연령대}_{일예산}_{메모(선택)}
```

**예시:** `LA_A2534_D5`

### 주요 타겟 세그먼트

| 코드 | 설명 |
|------|------|
| (클라이언트별 정의) | |

### 예산 단위

- 일예산 기준 만원 단위 (D5 = 일 5만원)

---

## 광고 네이밍 규칙

```
{VP코드}_{포맷}{비율}_{버전}_{메모(선택)}
```

**예시:** `VP1_IMG1x1_v1`
