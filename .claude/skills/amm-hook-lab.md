# /amm-hook-lab — 훅 문구 발산 및 선별

## 트리거

`/amm-hook-lab` 또는 "훅 발산", "훅 문구", "후킹 카피", "첫 줄 카피"

## 역할

하나의 소구점(VP)에서 훅 문구를 대량 발산한 후, ICE 채점으로 Top 10을 선별한다.
광고 소재·릴스·카드뉴스·블로그의 첫 줄 카피 재료를 빠르게 확보하는 전용 스킬.
Generator/Evaluator 분리 원칙 적용.

## Hook 게이트 (→ `.claude/hooks.md` 상세)

**Pre:** GATE-1 (클라이언트 확인)
**Post:** GATE-5 (산출물 저장 확인)

## 독립 실행

선행 스킬 없이 독립 실행 가능. VP 코드가 있으면 taxonomy.md에서 자동 로드.

## Sprint Contract (시작 전 합의)

스킬 실행 전 **AskUserQuestion 툴**로 아래를 순서대로 질문한다:

1. "어떤 클라이언트의 훅 문구를 발산할까요? (클라이언트명)"
2. "광고 계정명을 알려주세요. (예: main / kr-performance / default)"
3. "발산할 소구점을 알려주세요. (VP 코드 또는 핵심 메시지 한 줄)"
4. "타겟 페르소나는? (P 코드 또는 자유 기술)"
5. "사용할 채널은? (메타 광고 / 릴스 / 카드뉴스 / 블로그 / 전체)"
6. "목표 훅 개수는? (기본: 30개 / 많이: 60개 / 집중: 10개 심화)"

답변 확인 후 → Phase 1 (Generator)으로 진행.

## 실행 흐름

### Phase 0 — 브랜드·제품 로드 (GATE-0)

```
0. (GATE-0) docs/clients/{name}/brand-guide.md 읽기
   → 없으면: "/amm-brand-setup을 먼저 실행해주세요." → 중단
   → 제품 1개: 자동 선택, 이후 전체 진행에서 이 제품 정보 사용
   → 제품 2~3개: AskUserQuestion으로 선택 요청
     "이번에 광고할 제품/서비스를 선택해주세요: P01 {제품명} / P02 {제품명}"
   → 제품 4개 이상: products/ 폴더 목록 출력 → AskUserQuestion으로 선택 → P{N}.md 로드
```

### Phase 1 — 훅 발산 (Generator)

**이 단계에서는 평가 없이 생성만 한다.**

아래 6가지 유형별로 훅을 균등 발산한다:

| 유형 | 코드 | 예시 패턴 |
|------|------|---------|
| 페인포인트 | PAIN | "아직도 ~하고 있어요?" |
| 질문형 | QUES | "~해본 적 있나요?" |
| 수치/통계 | STAT | "XX%가 선택한 이유" |
| 사회증거 | SOC | "~하는 사람들이 공통적으로 말하는 것" |
| 감성 | EMO | "그때 그 기분, 기억하세요?" |
| 혜택 직접 | BENE | "~하면 ~가 달라집니다" |

발산 결과 형식:
```
## PAIN
1. ...
2. ...
3. ...
4. ...
5. ...

## QUES
...
```

> 발산 완료 후 → `.claude/rules/creative-design.md` "카피 정제 필수 패스" 적용 (Pass 1~3)
> 정제 완료 → `hook-candidates.md` 저장 → Phase 2 진입

### Phase 2 — ICE 채점 (Evaluator)

**Phase 1 파일 저장 후 별도 패스에서 수행.**

| # | 훅 문구 | 유형 | Impact | Ease | ICE | 선택 |
|---|---------|------|--------|------|-----|------|
| 1 | | | /10 | /10 | | ○/× |

**Impact 기준 (타겟 페르소나 기준):**
- 10점: 페르소나 100%가 멈춰서 읽을 것
- 5점: 절반 정도가 관심 가질 것
- 1점: 일부만 공감할 것

**Ease 기준 (제작 난이도):**
- 10점: 지금 바로 소재에 쓸 수 있음
- 5점: 약간의 수정·검증 필요
- 1점: 추가 크리에이티브·데이터 필요

### Phase 3 — Top 10 선별 및 활용 매핑

ICE 점수 상위 10개를 채널별로 매핑:

|------|---------|-----|---------|------------|
