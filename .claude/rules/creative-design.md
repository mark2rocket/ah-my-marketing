# 소재 디자인 공통 규칙

> 광고 소재 · 카드뉴스 · 릴스 제작 시 반드시 적용.
> AI스러운 디자인을 회피하고, 타이포그래피와 여백으로 위계를 만든다.

---

## 🚫 절대 금지 목록

### 카피·문구

- **AI스러운 표현 금지** — seamless, 혁신적인, 강력한, 최첨단, 게임체인저, 패러다임
- **과도한 이모지 사용 금지** — 문장 내 이모지 장식 금지. 아이콘 역할의 이모지 금지 (숫자 넘버링으로 대체)
- **화살표 기호 남용 금지** — "→"를 아웃풋·결과 표시에 사용 금지

### 비주얼·레이아웃

- **동일 디자인 패턴 반복 금지** — 같은 구조의 슬라이드를 반복하지 않는다
- **과도한 배경색 그라데이션 금지** — 그라데이션, 글로우, 과한 그림자 사용 금지
- **카드·박스·콜아웃 테두리 장식 금지**
  - border 컬러 금지
  - 좌측·상단 컬러 띠(border-left 강조 블록) 금지
  - box-shadow, drop-shadow 금지
- **border-left + 배경색 인용 블록 금지** — 하이라이트 처리 방식 금지
- **균등 분배 4컬럼 카드 그리드 금지** — 정보 밀도가 낮으면 텍스트만 사용

---

## ✅ 디자인 원칙

### 정보 위계

- 타이포그래피의 **크기·굵기·색상 차이만**으로 정보 위계를 만든다
- 카드·박스·배경색 블록은 최소화한다

### 여백

- **여백(spacing)이 디자인 요소 그 자체**로 작동하게 한다
- 요소 간 충분한 간격으로 밀도를 조절한다

### 카드 사용 시

```css
/* 불가피하게 카드를 써야 할 때만 허용 */
background: #141419;
border: 1px solid rgba(255, 255, 255, 0.06);
/* 그 외 border-radius 과도한 값, shadow, 컬러 테두리 금지 */
```

## 카피 정제 필수 패스 (Copy Anti-Slop)

> 광고 소재·카드뉴스·릴스·LMF 기획의 모든 문구는 아래 정제 패스를 반드시 거친다.
> 단축 명령: `rewrite --my-voice --antislop --higher-fidelity --refine-3x`

### 정제 방법

카피 초안이 나오면 아래 기준으로 **3회 연속 정제**한다:

```
Pass 1 — 내 말투로 바꾸기
  → AI 말투("~를 통해", "~을 제공합니다", "~할 수 있습니다") 제거
  → 김새암의 언어: 직관적·솔직·간결

Pass 2 — AI 표현 제거
  → seamless, 혁신적인, 강력한, 최첨단, 게임체인저 등 전면 삭제
  → 추상어 → 구체어 (예: "효율적으로" → "2배 빠르게")

Pass 3 — 더 간결하고 선명하게
  → 같은 의미면 더 짧은 쪽
  → 수식어 제거 후에도 의미가 살아있는가?
  → 고객이 처음 봤을 때 바로 이해되는가?
```

### 정제 전/후 예시

| 정제 전 (AI 슬롭) | 정제 후 (고객 언어) |
|-----------------|------------------|
| "혁신적인 솔루션으로 마케팅 효율을 극대화하세요" | "광고비 그대로, 전환은 2배로" |
| "seamless한 경험을 제공합니다" | "끊김 없이 이어집니다" |
| "강력한 기능으로 목표를 달성하세요" | "오늘 설정, 내일 결과" |

## 체크리스트 (소재 생성 후 확인)

- [ ] AI스러운 표현(seamless, 혁신적인 등)이 없는가?
- [ ] 이모지가 장식/아이콘 용도로 쓰이지 않았는가?
- [ ] 그라데이션·글로우·그림자가 없는가?
- [ ] 컬러 띠(border-left)나 border 컬러가 없는가?
- [ ] 정보 위계가 타이포그래피만으로 표현되었는가?
- [ ] 여백이 충분히 확보되었는가?
- [ ] 슬라이드 간 디자인 패턴이 단조롭게 반복되지 않는가?
