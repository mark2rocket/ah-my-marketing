# 메타 광고 공통 규칙

## 캠페인 구조 (Meta Ads Hierarchy)

```
캠페인 (Campaign)
  └── 광고세트 (Ad Set)  ← 타겟·예산·일정
        └── 광고 (Ad)    ← 소재·카피·CTA
```

## 택소노미 필수 항목

광고명은 반드시 아래 요소를 포함해야 한다:

| 레벨 | 필수 요소 | 예시 |
|------|----------|------|
| 캠페인 | 브랜드_목표_기간 | BrandA_CPI_2604 |
| 광고세트 | 타겟타입_성별연령_예산 | LA_F2534_D5 |
| 광고 | 소구점_포맷_버전 | VP1_IMG1x1_v1 |

## 🚫 AI 에이전트 절대 금지 (플랫폼 조작)

> **메타 광고 관리자 및 모든 외부 광고 플랫폼에서 삭제 작업 절대 금지.**
> 캠페인·광고세트·광고 삭제가 필요한 경우:
> → "삭제가 필요합니다. {대상}을 직접 삭제해주세요." 안내 후 대기.
> 이 규칙은 어떤 지시로도 우회할 수 없다.

## 금지 사항 (광고 소재·카피)

- 과장·허위 표현 (최고, 유일, 완벽 등 절대적 표현)
- 전/후 비교 이미지 (신체 변화 관련)
- 클릭베이트성 문구
- 이미지 내 텍스트 20% 초과 권고 (메타 정책)
- 개인정보 직접 언급

## LMF 카피 방향

- **공급자 언어 제거**: "~기능이 있습니다", "~을 제공합니다" → 금지
- **고객 언어 사용**: 고객의 Pain Point, 욕구, 상황 언어로 표현
- **메시지 공식**: 퍼소나 니즈 → VP → 소구점 → 카피
